使用场景

默认设置

可直接切到有线 / HDMI 或蓝牙音箱预设；如果不想手动指定，可恢复默认设置。

使用说明

蓝牙音箱通常要接通宿主的 PulseAudio 或 PipeWire 会话；有线 / HDMI 一般优先用 ALSA。

如果你不确定怎么填，可以先保持留空。

只有在下拉列表没有目标设备时，才建议手动输入 `plughw:1,0` 这类设备名。

蓝牙 / 桌面音频高级设置 可折叠

如果你接的是蓝牙音箱，通常需要宿主机提供 PulseAudio 或 PipeWire 会话；Docker 容器里仅透传 `/dev/snd` 往往只能驱动有线输出。
